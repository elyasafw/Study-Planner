# Study Planner

אפליקציית Node.js קטנה שמדמה בניית תוכנית לימוד קצרה ליום: חיפוש תלמיד, בניית תוכנית מתוך משימות לימוד, ושליחת תזכורת — תוך תרגול קוד אסינכרוני (Callbacks, Promises, async/await).

## הרצה

```bash
npm start
```

## מבנה הפרויקט

- `data.js` — נתוני התלמידים והמשימות.
- `callbacks.js` — חיפוש תלמיד לפי ID עם callback בתבנית error-first.
- `promises.js` — שליפת משימת לימוד לפי שם באמצעות Promise.
- `asyncAwait.js` — שליחת תזכורת לתלמיד עם `async/await` ו-`try/catch`.
- `main.js` — מריץ את כל הזרימה: חיפוש תלמיד קיים ולא קיים, בניית תוכנית ממשימה קיימת ולא קיימת, ושליחת תזכורות.

## מה קורה בהרצה

1. חיפוש תלמיד קיים ותלמיד שלא קיים.
2. שליפת משימת לימוד קיימת ולא קיימת, עם שרשרת `.then().catch().finally()`.
3. שליחת תזכורת לתלמיד עם מספר טלפון (מצליחה) ולתלמיד בלי מספר טלפון (נכשלת).
